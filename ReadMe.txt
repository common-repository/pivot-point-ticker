== Description ==

Displays pivot points for day trading stock markets. Pivot points are a calculation used by stock traders to identify support and resistance points during market hours.

This widget will retrieve the data and perform the calculation for any list of stocks or indices you add via comma separated lists of Yahoo format symbols. Eg: ^FTSE,^DJI,GOOG,BARC.L

The widget will fade between each set of pivot points. You can customize color, widths and titles to suit your webpage.

test out your design on our demo page here: http://www.livecharts.co.uk/pivot-point-ticker.php 


== Installation ==



1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto appearance > widgets and drag the widget to your sidebar


== Frequently Asked Questions ==

What symbols can I add?

Any stock or index in Yahoo data format. Eg: ^FTSE | ^DJI | GOOG | GE | BARC.L | VOD.L etc

Can I remove your link?

Yes, it is switched off by default but it would be nice to get some love.
